# Husn-e-No

A dark, aesthetic fashion website for custom hoodies and shirts.

## Features
- Admin upload panel
- Permanent product storage
- Firebase backend
- Cart & checkout system
- Domain-ready

## Tech Stack
- HTML, CSS, JavaScript
- Firebase Authentication
- Firestore Database
- Firebase Storage

## Hosting
Deploy using Firebase Hosting or connect a custom domain.
